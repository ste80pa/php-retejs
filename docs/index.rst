Welcome to PHP Retejs documentation!
===================================

**PHP Retejs** is a PHP implementation of the Engine for `ReteJS <https://github.com/retejs/rete>`_ inspired by `D3 Node Engine <https://github.com/retejs/cpp-engine>`_

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
